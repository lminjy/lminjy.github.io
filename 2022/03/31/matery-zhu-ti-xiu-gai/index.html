导航页面

在matery 主题中的_config.yml 可以修改菜单导航的名称、路径url和图标icon

  1. 在需要添加二级菜单的一级菜单下添加children关键字(如:About菜单下添加children)
  2. 在children下创建二级菜单的 名称name,路径url和图标icon.
  3. 注意每个二级菜单模块前要加 -.
  4. 注意缩进格式
  # 二级菜单写法如下
  Resources:
    icon: fas fa-list
    children:
      - name: Musics
        url: /musics
        icon: fas fa-music
      - name: Movies
        url: /movies
        icon: fas fa-film
      - name: Books
        url: /books
        icon: fas fa-book
      - name: Galleries
        url: /galleries
        icon: fas fa-image

新添加的菜单需要翻译成对应的中文，打开theme/next/languages/zh-CN.yml，设置：

Resources: 资源

安装插件

替换Hexo的代码高亮

cnpm i -S hexo-prism-plugin

修改根目录下 _config.yml 文件中 highlight.enable 的值为 false，并新增 prism 插件相关的配置

highlight:
  enable: false

prism_plugin:
  mode: 'preprocess'    # realtime/preprocess
  theme: 'tomorrow'
  line_number: false    # default false
  custom_css:

记得markdown代码框需要选择语言

设置内容搜索

cnpm install hexo-generator-search --save

在根目录下的 _config.yml 文件中，新增配置项：

search:
  path: search.xml
  field: post

中文链接转拼音

cnpm i hexo-permalink-pinyin --save

在根目录下的 _config.yml 文件中，新增配置项：

permalink_pinyin:
  enable: true
  separator: '-' # default: '-'

文章字数统计

cnpm i --save hexo-wordcount

在根目录下的 _config.yml 文件中，新增配置项：

postInfo:
  date: true
  update: true
  wordCount: true # 设置文章字数统计为 true.
  totalCount: true # 设置站点文章总字数统计为 true.
  min2read: true # 阅读时长.
  readCount: true # 阅读次数.

添加emoji表情支持

cnpm install hexo-filter-github-emojis --save

在根目录下的 _config.yml 文件中，新增配置项：

githubEmojis:
  enable: true
  className: github-emoji
  inject: true
  styles:
  customEmojis:

添加 RSS 订阅支持

cnpm install hexo-generator-feed --save

在根目录下的 _config.yml 文件中，新增配置项：

feed:
  type: atom
  path: atom.xml
  limit: 20
  hub:
  content:
  content_limit: 140
  content_limit_delim: ' '
  order_by: -date

执行 hexo clean && hexo g 重新生成博客文件，然后在 public 文件夹中即可看到 atom.xml 文件，说明你已经安装成功了。

音乐播放器

# Whether to display the musics.
# 是否在首页显示音乐.

  enable: true
  title: #非吸底模式有效
    enable: true
    show: 听听音乐
  autoHide: true    # hide automaticaly
  server: tencent   #require    music platform: netease, tencent, kugou, xiami, baidu
  type: playlist    #require song, playlist, album, search, artist
  id: 7071279876     #require    song id / playlist id / album id / search keyword
  fixed: true       # 开启吸底模式
  autoplay: true   # 是否自动播放
  theme: '#42b983'
  loop: 'all'       # 音频循环播放, 可选值: 'all', 'one', 'none'
  order: 'random'   # 音频循环顺序, 可选值: 'list', 'random'
  preload: 'auto'   # 预加载，可选值: 'none', 'metadata', 'auto'

  listFolded: true  # 列表默认折叠

Valine评论

在Valine官网 点击注册后创建新的应用，在应用–>设置–>应用凭证中获得对应的 AppId 和 AppKey ，在主题中的 _config.yml 文件，新增以下的配置项：

valine:
  enable: true
  appId: XXX
  appKey: XXX
  notify: true
  verify: false
  visitor: true
  avatar: 'retro' # Gravatar style : mm/identicon/monsterid/wavatar/retro/hide
  pageSize: 10
  placeholder: 'Comment here' # Comment Box placeholder
  background: /medias/comment_bg.png

最新版的 Valine 已经移除了邮件通知功能。确保 Valine 正常工作，可以配置相关插件。
